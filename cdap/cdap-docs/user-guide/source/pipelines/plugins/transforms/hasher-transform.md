# Hasher


Description
-----------
Hashes fields using a digest algorithm such as ``MD2``, ``MD5``, ``SHA1``, ``SHA256``, ``SHA384``, or ``SHA512``.


Configuration
-------------
**fields:** Specifies the fields to be hashed.

**hash:** Specifies the hashing algorithm.

- CDAP Pipelines Plugin Type: transform
