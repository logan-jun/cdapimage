==========
Indexsplit
==========

The INDEXSPLIT directive splits a string at a given index into two
substrings.

Deprecated
----------

Use the `SPLIT-TO-COLUMNS <split-to-columns.md>`__ or
`PARSE-AS-FIXED-LENGTH <fixed-length-parser.md>`__ directives instead.
